## Project Overview

This is a presentation research project analyzing **Kakao Corporation** (South Korea) for an MBA-level audience. The project focuses on innovation strategy, disruption analysis, and strategic recommendations.

## Project Structure

- `kakao-presentation.md` — 10-slide presentation content in Markdown (primary deliverable)
- `.claude/` — Claude Code configuration

## Context

- **Audience**: MBA professionals and students
- **Tone**: Business professional, market analysis
- **Focus**: Innovation types (product, operational, institutional) behind Kakao's success, potential disruptions, and strategic recommendations
- **Company**: Kakao Corporation (KOSPI: 035720), founded by Kim Beom-su, headquartered in Jeju City, South Korea
